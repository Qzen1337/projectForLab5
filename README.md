<br>
<h3>Функции</h3>
<ul>
  <li>FuncMenu(Меню пользователя)</li>
  <li>Input(Ввод данных)</li>
  <li>Processing(Обработка данных)</li>
  <li>Print(Вывод данных)</li>
  <li>Sum(Сумма)</li>
</ul>
